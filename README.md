# Foto by Eva - Portrait Photography

A premium, modern website for Foto by Eva, a luxury portrait photography studio in Virginia Beach.

## Features

- **Portfolio Showcase**: High-quality galleries for Maternity, Newborns, Families, and Women.
- **Interactive Lightbox**: View images in full resolution.
- **Studio Information**: Details about the studio, investment, and testimonials.
- **Contact System**: Personalized contact form for booking sessions.
- **Responsive Design**: Optimized for all devices with smooth animations.

## Technologies Used

- **Framework**: [Vite](https://vitejs.dev/) + [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Building for Production

To create an optimized production build:

```sh
npm run build
```

The output will be in the `dist` folder.
